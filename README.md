# learning-langchain

A hands-on repository for learning and experimenting with **LangChain** to build LLM-powered applications using Python and JavaScript/TypeScript.

## 📌 Goals

* Understand LangChain core concepts
* Build real-world LLM workflows
* Explore agents, chains, tools, and memory
* Practice prompt engineering and RAG patterns

## 🧰 Tech Stack

* LangChain
* OpenAI API
* Python / Node.js (JavaScript & TypeScript)
* Vector stores (FAISS, Chroma, etc.)

## 📂 Project Structure

```
learning-langchain/
│
├── examples/        # Small focused examples
├── projects/        # Mini real-world projects
├── notes/           # Personal learning notes
└── README.md
```

## 🚀 Getting Started

### 1️⃣ Clone the repo

```bash
git clone https://github.com/your-username/learning-langchain.git
cd learning-langchain
```

### 2️⃣ Install dependencies

#### Python

```bash
pip install -r requirements.txt
```

#### Node.js

```bash
npm install
```

### 3️⃣ Set environment variables

Create a `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```
